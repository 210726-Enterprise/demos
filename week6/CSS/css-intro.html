Intro to CSS

CSS can be included in 3 different ways:

  1. Inline Styling with the style attribute. This is not preferred because it's not maintainable.
  2. Internal stylesheets using the style element in the head of our html page
  3. External stylesheets using the link element in the head of our html page

CSS rules are defined by key value pairs, separated by a colon ( : ).
