Generic (Structural) Markup Tags

The elements that are used to present the content itself, with no extra or inferred meaning.

Semantic Markup

Provides extra information about the content enclosed. Such as, where to place emphasis, or if something is being quoted, or the meaning of an acronym. Semantic elements make it easier for search engine crawlers, assistive technologies like screen readers, and web developers to interpret your web page, and are much tidier than a maze of meaningless

and tags.
